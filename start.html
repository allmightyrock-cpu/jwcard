🗺️ 설치, 이렇게 진행돼요

코딩을 몰라도 괜찮습니다. 순서대로 따라하면 누구나 설치할 수 있어요.
전체 그림을 먼저 가볍게 훑어보세요.

💰 무료 ⏱️ 약 30~40분 🔒 개인정보 미수집 📱 휴대폰 앱(PWA)
준비물: 구글 계정 1개, 인터넷이 되는 PC.
어려운 코딩은 없습니다 — 대부분 "콘솔에서 값 복사 → 붙여넣기 → 업로드" 입니다. 중간에 막히면 설치 점검(자동 진단)이 무엇이 문제인지 알려줍니다.
전체 6단계
1
📥 파일 받기 약 2분
GitHub에서 최신 파일 묶음을 내려받습니다. (압축 풀기)
2
🔥 Firebase 만들기 약 10분
무료 데이터 저장소(Firebase)를 만들고, 콘솔에서 보이는 설정값을 복사해 둡니다. 익명 로그인도 켭니다.
자세한 방법 보기
3
⚙️ 설정 만들기 약 3분
복사한 값을 간편 설치 도우미에 붙여넣기만 하면 config.js 파일이 자동으로 만들어집니다. (직접 코드 편집 불필요)
⚙️ 간편 설치 도우미 열기
4
☁️ 배포(업로드) 약 10분
받은 config.js를 폴더에 넣고, Cloudflare Pages에 폴더를 통째로 올리면 접속 주소가 발급됩니다.
배포 방법 보기
5
🩺 점검하기 약 2분
발급된 주소 뒤에 /diagnostics.html을 붙여 열고 점검 시작을 누르면, 연결·지도·앱 상태를 자동으로 확인해 줍니다. 문제가 있으면 해결 방법도 알려줍니다.
🩺 설치 점검 열기
6
✅ 사용 시작 약 5분
/admin.html 접속 → 최초 접속 시 관리자 비밀번호 설정 → 전도인 등록 → (있다면) 엑셀로 구역 데이터 업로드. 끝!
🎉 이게 전부입니다
한 번 설치해 두면, 전도인은 휴대폰에서 앱처럼 쓰고(홈 화면 추가), 관리자는 PC에서 구역을 배정·관리합니다. 업데이트가 있으면 앱에 배너로 안내됩니다.
중간에 막히거나 오류가 나면 — 먼저 설치 점검, 그다음 자주 묻는 질문을 확인하세요.
