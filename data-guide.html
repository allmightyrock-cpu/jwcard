전자 구역카드 시스템 · 관리자용

구역 데이터 입력 가이드

프로그램 설치가 끝났다면, 이제 우리 회중의 구역(영역)과 세대 주소를 시스템에 입력해야 사용할 수 있습니다.
이 문서는 구역카드를 만들고 주소 데이터를 넣는 방법을 처음부터 안내합니다.

목차 ① 전체 흐름 ② 구역카드 이해 ③ 직접 입력(한 장씩) ④ 엑셀로 입력 ⑤ 한 번에 여러 구역 ⑥ 세대 목록 꾸미기 ⑦ 지역명·정비 ⑧ 마무리 체크리스트

1 전체 흐름

설치 → 데이터 입력(이 문서) → 인도자 운용, 세 단계입니다.

  1. 설치 — config.js 작성·배포 (별도 "설치 안내서" 참고). 이미 끝났다고 가정합니다.
  2. 구역 데이터 입력 — 우리 회중의 구역과 세대 주소를 넣습니다. ← 지금 이 단계
  3. 운용 — 인도자가 구역을 배정하고 전도인이 방문 결과를 기록합니다. ("인도자 교육 가이드" 참고)
💡누가 하나요? — 구역 데이터 입력은 관리자 또는 봉사감독자 권한으로 admin.html(관리자 화면) → 구역관리에서 합니다.
⚠먼저 연습하세요. 실제 구역을 넣기 전에, 샘플(연습용) 구역카드를 한 장 만들어 입력·수정·삭제를 익혀 보면 실수를 줄일 수 있습니다.

2 구역카드란?

구역카드 1장 = 한 구역(영역)의 봉사 단위입니다. 아래 정보로 이루어집니다.

구성 요소설명예시
구역 번호정수만 사용 (1, 2, 3 …). 회중에서 부르는 구역 번호.1 / 2 / 145
구역 이름알아보기 쉬운 이름. 아파트 단지명·동네명 등.○○아파트 1단지, △△동 주택가
구역 유형(그룹)주택·상가·아파트·격지·인터폰·편지 등으로 분류. 필터·통계에 사용.아파트 구역
세대(호수) 목록그 구역에 속한 집들의 주소 목록. 도로명·번지·건물명·세부주소(호수).행복로 12, 101동 1001호 …
💡구역 유형(그룹)은 설정 → 구역그룹관리에서 회중에 맞게 추가·이름변경할 수 있습니다.

3 방법 A — 직접 입력(한 장씩)

구역 수가 적거나, 한 장을 새로 만들거나 수정할 때 좋습니다.

  1. 구역관리 화면에서 + 구역 추가 클릭
  2. 구역 번호(정수)·구역 이름·구역 유형을 입력하고 저장
  3. 방금 만든 카드의 ⋮ → 카드 팝업이 열리면 편집
  4. 편집 모달에서 호수 목록 탭으로 이동
  5. 각 세대를 한 줄씩 추가 — 도로명 · 번지 · 건물명 · 세부주소(호수) 입력
  6. 다 입력했으면 저장. 세대 수가 자동 집계됩니다.
💡세대가 많은 구역은 다음 ④ 엑셀 입력이 훨씬 빠릅니다. 직접 입력은 소규모·수정용으로 쓰세요.

4 방법 B — 엑셀로 입력

세대가 많은 구역을 빠르게 넣을 때 가장 효율적입니다. 엑셀(xlsx) 한 파일 = 구역 한 장.

① 파일 이름 규칙

{구역번호}_{구역이름}.xlsx 형식으로 저장합니다.

  • 예) 1_행복아파트.xlsx, 145_중앙로 상가.xlsx
  • 구역번호는 정수만 가능합니다. 001, 13-1, 23.5 형식은 인식되지 않습니다.

② 열(칸) 순서 — 정확히 지켜 주세요

엑셀 첫 행부터 아래 순서로 칸을 채웁니다. 한 행 = 한 세대입니다.

열123456
항목도로명번지건물명세부주소금지(Y)메모
예시행복로12행복아파트101동 1001호경비실 옆
  • 도로명·번지: 도로명주소 기준. (지번만 있으면 번지 칸에 지번을 넣어도 됩니다)
  • 건물명: 아파트·빌딩 이름. 없으면 비워도 됩니다.
  • 세부주소: 동·호수 등 실제 방문 단위. (예: 101동 1001호, 2층 201호)
  • 금지(Y): 방문 금지 세대면 Y. 일반 세대는 비워 둡니다.
  • 메모: 참고사항(이유 없는 간단 메모). 개인정보·민감내용은 적지 않습니다.

③ 업로드

  1. 구역관리 → 엑셀 업로드 클릭
  2. 위 규칙으로 만든 .xlsx 파일 선택
  3. 미리보기로 주소가 잘 들어갔는지 확인 후 저장
⚠빈 행(완전히 비어 있는 줄)은 세대로 인식되지 않습니다. 줄 사이 빈 칸을 피해 주세요.

5 방법 C — 한 번에 여러 구역(일괄 업로드)

처음 회중 데이터를 통째로 구축할 때 가장 빠릅니다.

  1. 각 구역을 {번호}_{이름}.xlsx 형식으로 여러 개 준비 (④의 규칙과 동일)
  2. 구역관리 → 📦 일괄 업로드 클릭
  3. 여러 xlsx 파일을 한꺼번에 선택하거나, ZIP으로 묶어 업로드
💡이미 존재하는 구역번호는 자동으로 건너뜁니다. 그래서 나중에 새 구역만 추가 업로드해도 기존 데이터가 덮어써지지 않습니다.

6 세대 목록 꾸미기 (선택)

현장에서 보기 좋게, 안전하게 봉사하도록 도와주는 표시들입니다. 편집 모달 → 호수 목록에서 설정합니다.

구분선 · 지시문

  • 구분선: 세대 목록을 묶음으로 나눕니다. (예: "1·2라인 진입", "3·4라인 진입") — 행 오른쪽 ─ 버튼 또는 하단 구분선 추가.
  • 지시문: 구분선 행의 + 지시문 버튼으로 여러 줄 안내를 넣습니다. (공동현관 비밀번호·주의사항 등) → 전도인 앱에 황색 박스로 표시됩니다.
  • 구분선은 세대 수·진행률 계산에서 제외됩니다.

형제 동반 필수 표시 ⚠

안전을 위해 동반이 필요한 세대는 호수 목록에서 형제동반필수를 체크합니다. 전도인 앱에 ⚠ 경고가 뜨고, 자매 단독·자매끼리 배정이 금지됨을 안내합니다. (형제 또는 형제가 포함된 부부·친족 짝으로만 배정)

7 지역명 설정 · 데이터 정비

지역명(지도 검색 기준)

설정에서 시·도 단위 기본 지역명을 넣어 두면, 도로명 주소를 지도에서 정확히 찾습니다. (예: 기본 지역명 "○○시" → "행복로 12"를 "○○시 행복로 12"로 검색)

건물명 오류 일괄 정비

엑셀로 가져온 뒤 도로명·번지가 바뀌는 행에 이전 건물명이 잘못 이어졌다면, 설정 → 🔧 데이터 정비 → 건물명 오류 일괄 정비로 한 번에 고칠 수 있습니다.

8 마무리 체크리스트

  • ☐ 연습용 샘플 구역으로 추가·수정·삭제를 익혔다
  • ☐ 구역 유형(그룹)을 회중에 맞게 정리했다 (설정 → 구역그룹관리)
  • ☐ 지역명(시·도)을 설정했다
  • ☐ 엑셀 파일명을 {번호}_{이름}.xlsx로 맞췄다 (번호는 정수)
  • ☐ 열 순서(도로명·번지·건물명·세부주소·금지·메모)를 지켰다
  • ☐ 일괄 업로드 후 세대 수·주소가 맞는지 카드 몇 개를 확인했다
  • ☐ 형제 동반 필수 세대를 표시했다
  • ☐ 연습용 샘플 구역카드는 삭제했다
✅데이터 입력이 끝나면 이제 인도자 교육 가이드를 따라 배정·운용을 시작하면 됩니다.
