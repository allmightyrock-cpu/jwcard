STEP 1 · 붙여넣기

Firebase 설정값 붙여넣기

Firebase 콘솔 → ⚙ 프로젝트 설정 → 일반 → 내 앱 → SDK 설정에서 보이는 내용을 그대로 복사해 아래에 붙여넣으세요. (import…, const firebaseConfig…가 섞여 있어도 괜찮아요 — 알아서 값만 골라냅니다.)

붙여넣은 코드는 저장만 자동으로 골라낼 뿐, 그대로 config.js에 들어가지 않습니다. 안심하고 통째로 붙여넣으세요.
