전체 공지 (주황 배너) 회중 전체에 공지된 사항입니다. 구역 공지와 별개로 표시됩니다.

🔄 부재방문 단계
🔄
부재방문이란? 호별방문에서 만나지 못한 세대만 골라서 다시 방문하는 단계입니다. 회중 설정에 따라 진행합니다.
🔵
부재방문 단계 진입 시 화면 변화 카드 상단에 노란 단계 배지가 표시됩니다(예: 3회차 부재1). 이미 만난 집은 흐리게(회색) 표시되어 탭이 되지 않습니다. 만나지 못한 집만 활성화되어 방문합니다.
➡️
단계 진행 방법 호별방문 완료 후 하단 "✓ 호별 완료 → 부재1" 버튼을 탭하면 부재방문 단계로 넘어갑니다. 다음날 카드를 열면 자동으로 부재방문 단계로 전환됩니다.
✅ 완료 처리
✓
구역 완료 신청 모든 단계 방문 후 하단 "✓ 구역 완료" 버튼을 누릅니다. 관리자가 확인 후 최종 처리하면 다음 회차가 시작됩니다.
⏸
미완료로 마치기 오늘 활동을 마쳐야 할 때 "⏸ 미완료 마치기"를 누릅니다. "여기까지" 표시 후 활성화됩니다.
🌙
자동 완료 (자정) 봉사를 한 카드(1집 이상 체크)는 매일 자정에 자동으로 완료되어 다음 봉사 준비 상태가 됩니다. 계속 진행하려면 "미완료"로 마치거나 책갈피 표시를 해두세요.
🗺 지도
📌
네이버 지도 자동 표시 구역 첫 번째 주소를 자동으로 지도에 표시합니다. 각 건물에 번지가 핀으로 표시됩니다.
🔍
"지도에서 찾기" 버튼 자동 표시가 되지 않을 경우 버튼을 눌러 직접 검색할 수 있습니다.
🧭 길 안내 모드
🧭
길 안내 모드 진입 구역카드 상단 헤더 우측 [🧭 길 안내] 버튼을 탭하면 길 안내 전용 화면으로 전환됩니다.
📋
전체 방문 목록 구역 내 모든 세대가 번지수 중심으로 목록에 표시됩니다. 방문 완료된 세대는 흐리게, 현재 대상은 파란색으로 강조됩니다.
🗺️
주소 탭 → 지도 앱 경로 안내 목록에서 방문할 주소를 탭하면 앱 선택 팝업이 뜹니다. T맵·네이버지도·카카오맵 중 하나를 선택하면 해당 주소로 가는 실시간 경로 안내가 시작됩니다.
🚶
도보 / 자동차 모드 팝업 상단 세그먼트 버튼으로 도보 또는 자동차 경로를 선택합니다. 선택한 이동 수단으로 지도 앱 경로 안내가 시작됩니다.
✅
방문 완료 기록 상단 카드의 [✅ 방문 완료] 버튼을 탭해 결과를 선택합니다. 완료 후 자동으로 다음 미방문 세대로 이동합니다.
↩
구역카드로 돌아가기 화면 하단 고정 바의 [← 구역카드로] 버튼을 탭하면 언제든 원래 구역카드 화면으로 돌아올 수 있습니다.
✉️
호수 미기록 수정 요청 세대 목록에서 「호수 미기록 ✉」 배지를 탭하면 인도자/관리자에게 보낼 수정 요청 메시지가 자동 작성됩니다. 카카오톡·문자 등 원하는 앱으로 전송하세요.

👥 권한별 기능
👤
일반 전도인 봉사 참여 신청 → 자동 배정 알림 → 구역카드 열람·방문 기록. PIN 불필요.
📋
인도자 — 배정 패널 PIN 로그인 후 📋 버튼 → 구역 배정 탭에서 구역을 검색·필터하여 전도인에게 배정합니다. 카카오톡으로 구역 공유도 가능합니다.
📋 인도자 배정 3단계
① 출석 확인 — 신청 전도인 실시간 목록 확인
② 구역 선택 — 정렬·필터·검색으로 오늘 사용할 카드 선택
③ 배정 확정 — 전도인을 선택하고 배정 확정. 전도인 화면에 번호 즉시 표시

봉사감독자 구역카드 비활성화·활성화 최종 권한을 가집니다. 회차제한에 도달해 비활성화된 카드도 재활성화할 수 있습니다.
💬 자주 묻는 질문
❓
앱 설치가 필요한가요? 아니요. 스마트폰 브라우저(크롬, 사파리)에서 바로 사용합니다.
❓
참여 신청 후 구역 번호가 안 보여요 인도자가 아직 배정 중입니다. "⏳ 인도자 배정 중..."이 나오면 잠시 기다리세요. 배정되면 번호가 자동으로 나타납니다.
❓
카드가 회색으로 잠겨 있어요 회차제한에 도달해 자동 비활성화된 카드입니다. 봉사감독자 또는 관리자에게 재활성화를 요청하세요.
❓
부재방문 단계에서 집 탭이 안 돼요 이전 방문에서 이미 만난 집(응답·거절 등)은 부재방문 단계에서 비활성화됩니다. 회색으로 표시된 집은 탭이 되지 않으며, 이는 정상입니다.
❓
인터넷이 없으면 사용 불가한가요? 네. 방문 결과 저장과 실시간 배정 알림은 인터넷 연결이 필요합니다.
❓
비밀번호를 잊어버렸어요 개인 PIN 방식: 관리자에게 PIN 초기화를 요청하세요. 초기화 후 다음 로그인 시 새 PIN을 설정합니다.
공용 암호 방식: 관리자에게 새 암호를 문의하세요.
❓
다른 전도인의 기록이 보여요 같은 구역에 여러 명이 배정된 경우 서로의 방문 기록이 실시간으로 보입니다. 이는 정상입니다.

구역 카드마다 + 전도인 추가를 눌러 배정할 전도인을 선택하세요.
🗓 오늘의 봉사 탭 — 3단계 배정 흐름
1
① 출석 확인
전도인이 앱에서 "✋ 오늘의 봉사 참여 신청"을 누르면 출석 목록에 실시간으로 나타납니다.
앱 사용이 어려운 전도인은 이름 직접 추가로 수동 등록하세요.
2
② 구역 선택
오늘 사용할 구역카드를 탭해서 선택합니다. 여러 개 선택 가능합니다.
오늘 예정 구역이 미리 설정된 경우 상단에 자동으로 표시됩니다.
3
③ 전도인 배정
각 구역 카드 아래 + 전도인 추가를 탭해서 배정할 전도인을 선택합니다.
모두 선택 후 ✅ 배정 확정 버튼을 누르면 전도인 화면에 구역번호가 즉시 표시됩니다.
💡 배정이 완료되면 전도인 앱에 "🗺 구역카드 바로 열기" 버튼이 나타납니다. 전도인에게 앱을 확인하도록 안내하세요.
👥 배정 인원 규칙
👥
카드 당 최대 6명 한 구역카드에 최대 6명까지 배정할 수 있습니다.
✅
공식 배정 — 앞 2명 1·2번째 전도인이 공식 배정자로 카드 헤더에 표시됩니다.
🔵
동행 — 나머지 최대 4명 3번째부터는 "함께 활동중" 파란 점(🔵)으로 구분 표시됩니다. 구역 목록에서도 "+N명 동행"으로 표시됩니다.
📱
카카오톡으로 구역 공유 구역 목록에서 구역 번호(숫자)를 탭하면 카카오톡으로 구역 정보와 접속 링크를 공유할 수 있습니다.
📅 오늘 예정 구역 — 배정 탭 상단
📅
자동 표시 관리자가 오늘 요일에 배분한 구역이 배정 탭 상단에 자동으로 나타납니다. 탭하면 바로 배정 패널이 열립니다.
↩
반납 버튼 예정 구역 오른쪽 반납 버튼을 탭하면 해당 구역이 미할당 풀로 즉시 반납됩니다. 현장에서 사용하지 않을 구역을 수동으로 돌려줄 때 사용합니다.
🌙
자정에 자동 반납 체크 시(기본값) 자정이 지난 후 앱 접속 시 해당 요일 배분 카드가 자동으로 미할당 풀로 반납됩니다. 해제하면 다음 주도 그대로 유지됩니다.
➕
미할당 카드 현장 추가 예정 구역 섹션 하단 ▼ 목록 보기를 탭하면 아직 어느 요일에도 배분되지 않은 카드를 검색해 오늘 일정에 즉시 추가할 수 있습니다. 카테고리 필터 칩으로 범위를 좁혀 빠르게 찾을 수 있습니다.
🔍 구역 배정 탭 — 검색 및 필터
↕
정렬 예정 우선 / 오래된 순 / 최근 완료 다음 / 세대 적은 순 / 진행률 낮은 순
🏷
유형·상태·회차 필터 구역 유형(주택구역, 상가구역 등), 배정 상태(미배정만·전체·미완료만), 회차(1~6회차)로 각각 필터합니다.
🔘
칩 필터 (4가지) 📅 오늘 예정만 — 오늘 요일에 배분된 구역만 표시 (기본 켜짐)
⚠ 동행필수 없는 구역만 — 혼자 방문 가능한 구역만
⏰ 재방문 세대 있는 구역 — 재방문 코드가 기록된 세대가 있는 구역
🕐 6개월+ 미사용 — 최근 6개월간 배정 이력이 없는 구역

🔄 부재방문 단계 안내
🔄
부재방문이란? 호별방문에서 만나지 못한 집만 골라서 다시 방문하는 단계입니다. 회중 설정에 따라 1~2회 진행합니다.
⚙
설정 위치 관리자 페이지 → 설정 → 방문횟수에서 "호별만 / 부재 1회 / 부재 2회" 중 선택합니다.
💡 부재방문 중인 구역은 칩 필터 "🔄 부재방문 중"으로 빠르게 찾을 수 있습니다.
⭐ 봉사감독자 전용 기능
🔒
구역 비활성화 / 활성화 구역 비활성화 탭에서 특정 구역을 비활성화하거나, 회차제한으로 자동 비활성화된 구역을 다시 활성화할 수 있습니다. 이 권한은 봉사감독자만 가집니다.
⚠ 구역 비활성화는 해당 구역에 접속 중인 전도인에게 즉시 적용됩니다. 신중하게 사용하세요.

비활성화된 구역카드는 전도인이 열면 비활성 화면이 표시됩니다.
오직 봉사감독자만 다시 활성화할 수 있습니다.
