전자 구역카드 시스템 · 인도자용

인도자 교육 가이드

호별봉사·전시대봉사 인도자가 시스템을 운용하고 전도인을 안내하기 위한 단계별 안내서입니다.
이름·암호 외에 별도 설치나 가입은 필요하지 않습니다.

목차 ① 공통 기초 ② 호별봉사 인도자 ③ 전시대봉사 인도자 ④ 전도인에게 안내할 것 ⑤ 자주 묻는 질문

1 공통 기초

먼저 시스템의 큰 그림과 인도자 진입 방법을 익힙니다.

시스템 한눈에 보기

  • 전도인은 이름(또는 구역번호)으로 들어와 배정받은 구역카드를 열고 방문 결과를 기록합니다.
  • 인도자는 "배정 패널"에서 오늘 쓸 구역을 고르고 전도인에게 구역을 배정합니다.
  • 배정 결과는 실시간으로 전도인 화면에 구역 번호로 나타납니다. (종이·전화 없이 현장에서 바로)

인도자 진입 방법

  1. 앱 로그인 화면에서 이름을 입력합니다.
  2. 배정 패널만 열기 버튼을 누릅니다. (구역카드 화면에서는 하단 📋 버튼으로도 들어옵니다)
  3. 암호를 입력합니다 — 회중 설정에 따라 둘 중 하나입니다.
    • 개인 PIN 방식: 처음 한 번 본인 PIN(4자리 이상)을 직접 정하고, 이후 그 PIN으로 들어옵니다.
    • 공용 암호 방식: 관리자가 정한 하나의 암호를 모든 인도자가 공유해 사용합니다.
💡팁 — 인도자 패널 상단에 구역 배정 / 오늘의 봉사 / 사용법 탭이 보이면 정상 진입된 것입니다. (봉사감독자는 구역 비활성화 탭이 추가로 보입니다.)

역할별 차이 (인도자가 알아둘 것)

역할진입할 수 있는 일
전도인이름/구역번호배정받은 구역카드 열람·방문 기록
인도자 · 구역의종이름 + 암호호별봉사 구역 추가·배정·반납, 전시대봉사 팀 편성
봉사감독자이름 + 암호인도자 기능 + 봉사일정 구역 배분 + 구역 비활성화/활성화
관리자이름 + 암호전체 기능 + 설정·권한 관리

2 호별봉사 인도자

"오늘 쓸 구역 준비 → 현장에서 출석 받고 배정"의 두 단계로 운용합니다.

준비 — 오늘 쓸 구역 담기 (구역 배정 탭)

  1. 구역 배정 탭을 엽니다. 관리자가 그 요일에 미리 배분해 둔 구역이 초록 카드로 보입니다.
  2. 구역이 부족하면 ➕ 구역 추가를 눌러 팝업에서 구역을 찾습니다.
    • 드롭다운: 상태(미배정/전체/미완료)·유형(아파트·상가·주택)·회차·정렬(오래된 순 등)
    • 칩 필터: ⚠ 동행필수 없는 구역만 · ⏰ 재방문 세대 있는 구역 · 🕐 6개월+ 미사용
    • 검색: 구역 번호·이름·회차로 바로 찾기
  3. 원하는 구역의 + 추가를 누르면 오늘 배당 목록에 들어갑니다.
🗓미리 배당해 두기 (봉사감독자·관리자) — 봉사감독자나 관리자는 관리자 화면 → 봉사일정 탭에서 요일별로 구역을 미리 배분할 수 있습니다(자동 배분 또는 검색해서 직접 배분). 이렇게 배분해 두면 그 요일 인도자 화면에 자동으로 초록 카드(배당된 구역)로 나타나, 현장에서 바로 사용할 수 있습니다. 인도자는 부족한 구역만 ➕ 구역 추가로 더하면 됩니다.
🔁자동 반납 — "자정에 자동 반납"이 켜져 있으면 그날 쓴 구역은 자정에 자동으로 미할당으로 돌아갑니다. 다음 주에도 같은 구역을 유지하려면 이 체크를 해제하세요.

현장에서 — 오늘의 봉사 3단계

모임 장소에서 오늘의 봉사 탭을 열고 ① 출석 → ② 구역 선택 → ③ 배정 순으로 진행합니다.

  1. ① 출석 확인 — 전도인이 자기 폰에서 ✋ 오늘의 봉사 참여 신청을 누르면 즉시 출석 목록에 뜹니다. 신청 못 한 사람은 이름을 직접 입력해 추가할 수 있습니다.
  2. ② 구역 선택 — 준비 단계에서 담아둔 구역들 중 오늘 쓸 구역을 탭해 고릅니다. (여러 개 가능)
  3. ③ 배정 — 각 구역 카드에서 전도인을 선택하고 ✅ 배정 확정을 누릅니다. 확정하면 전도인 화면에 구역 번호가 자동으로 나타납니다.

전도인 배정 패널 사용법 (빠른 배정)

구역 배정 탭에서 구역 카드를 누르면, 그 카드 바로 아래에 전도인 선택 패널이 펼쳐집니다.

  • 오늘 참여 신청한 전도인이 맨 위에 강조(✋)되어 나옵니다 — 보통 이분들부터 배정합니다.
  • 그 아래 그 외 전도인, 맨 아래 이미 배정됨 순서입니다.
  • 이름이 많으면 패널 위 🔍 이름 검색에 두 글자만 쳐도 바로 좁혀집니다.
  • 이름을 누르면 선택, 다시 누르면 취소됩니다(여러 명 가능). 한 구역에 최대 6명 — 앞 1·2번째가 공식 배정, 3번째부터는 "함께 활동중"으로 표시됩니다.

배정 변경·구역 반납

  • 전도인 바꾸기: 패널에서 선택을 해제/추가하고 다시 배정 확정.
  • 구역 반납: 구역 카드의 반납 버튼을 누르면 그 구역이 미할당으로 돌아갑니다.
  • 변경·반납은 전도인 화면에 실시간 반영됩니다.
⚠이미 방문 기록이 있는 구역을 회수할 때는 기록이 보존되며 "미완료" 상태로 남습니다. 전도인이 회수 사실을 모르고 방문하지 않도록, 회수 시 전도인에게도 알려 주세요.

3 전시대봉사 인도자

전도인의 요일 신청을 받아 팀을 편성하고 확정합니다.

전도인은 이렇게 신청합니다

  • 이름을 입력하면 메인 화면에 전시대봉사 일정이 나타납니다.
  • 참여할 요일 토글을 켜면 신청됩니다.
  • 신청·취소는 봉사 시작 3시간 전까지 자유롭게 가능합니다.

인도자의 팀 편성

  1. 전시대봉사 관리(팀 편성) 화면으로 들어갑니다.
  2. 요일별 신청자 명단을 보고 팀(A·B·C…)과 슬롯(인원)에 전도인을 배치하고, 필요하면 팀 담당자를 지정합니다.
  3. 편성을 마치면 확정합니다. 상태가 팀 편성중 → 팀 확정으로 바뀝니다.

마감 전후 / 전도인 화면

  • 마감 전(편성중): 전도인은 신청·취소 가능, 인도자는 실시간으로 신청 현황을 봅니다.
  • 마감 후(시작 3시간 이내): 새 신청은 막히고, 필요한 경우 취소 요청만 가능합니다.
  • 확정 후: 전도인은 "내 배정"에서 팀·시간·동반자·장소를 확인합니다.
💡팁 — 편성 중에도 신청이 들어올 수 있으니, 마감 시간 직전에 명단을 한 번 더 확인하고 확정하면 좋습니다.

4 전도인에게 안내할 것

인도자가 전도인에게 알려주면 좋은 핵심 사용법입니다.

  • 참여 신청: 모임 장소 도착 → 이름 입력 → ✋ 오늘의 봉사 참여 신청.
  • 구역 받기: 배정되면 구역 번호가 자동 표시 → 🗺 구역카드 바로 열기. (직접 번호+이름으로 열 수도 있음)
  • 방문 결과: 세대 옆 ✅ 방문 완료 → 결과 선택(대화함·아무도 없음 등) → 다음 세대로 자동 이동.
  • 길 안내: 🧭 길 안내 → 주소 탭 → 지도 앱(T맵·네이버·카카오) 선택 → 안내 시작.
  • 부재 방문: 호별 1회전을 마치면 ✓ 호별 완료 → 부재1로 전환해 부재 세대만 다시 돕니다.
  • 구역 완료: 모두 마치면 ✓ 구역 완료 신청 → 인도자/관리자가 확인 처리합니다.
📖앱 안의 "사용법"에 더 자세한 설명서가 있습니다. 처음 쓰는 전도인에게는 이 설명서를 한 번 같이 보여주면 좋습니다.

5 자주 묻는 질문

전도인 화면에 구역 번호가 안 떠요

아직 배정이 확정되지 않았을 수 있습니다. 인도자가 ✅ 배정 확정을 눌렀는지 확인하세요. 확정 후에는 전도인 화면에 자동으로 나타납니다.

오늘의 봉사 ② 구역 선택에 구역이 안 보여요

먼저 구역 배정 탭에서 ➕ 구역 추가로 오늘 쓸 구역을 담아야 ② 단계에 나타납니다.

PIN을 잊었어요

개인 PIN 방식이라면 관리자에게 초기화를 요청하세요. 공용 암호 방식이라면 관리자가 공유한 암호를 다시 확인하세요.

지난주 구역을 계속 쓰고 싶어요

구역 배정 탭의 "자정에 자동 반납" 체크를 해제하면 그 구역이 다음 주에도 유지됩니다.

자매들에게 안전한 구역만 주고 싶어요

구역 추가 팝업에서 "⚠ 동행필수 없는 구역만" 칩을 켜면 동행필수 표시가 없는 구역만 골라 담을 수 있어, 자매들에게 안전한 구역 위주로 배정할 수 있습니다.
